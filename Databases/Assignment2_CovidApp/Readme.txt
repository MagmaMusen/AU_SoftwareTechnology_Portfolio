1. Ændre connection string til at passe med din database
2. Open package manager console
3. Vær sikker på at table folders i database og migration folder i vs begge er tomme.
4. Skriv "Add-Migration InitialCreate"
5. Skriv "Update-Database"
6. Kør programmet med ctrl+f5 for at visualisere data

Vores dummy data genereres randomly inde i data context filen.
